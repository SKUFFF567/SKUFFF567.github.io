# RecLamBot
RecLamBot — Автопостинг VK и Telegram
Сервис для автоматической публикации рекламных постов в группы ВКонтакте и Telegram-каналы.

Возможности:
• Автопостинг на стены групп VK
• Отправка в Telegram (бот и userbot режимы)  
• Настраиваемый кулдаун между постами
• Статистика отправок
